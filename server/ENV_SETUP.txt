Environment Variables Setup
============================

1. Create a file named .env in the server folder
2. Add the following content:

GMAIL_USER=your-email@gmail.com
GMAIL_APP_PASSWORD=your-16-character-app-password
PORT=5000
OPENAI_API_KEY=your-openai-api-key-here
OPENAI_MODEL=gpt-3.5-turbo

Gmail App Password Setup:
1. Go to https://myaccount.google.com/security
2. Enable 2-Step Verification if not already enabled
3. Go to https://myaccount.google.com/apppasswords
4. Select "Mail" and "Other (Custom name)"
5. Enter "Event Manager" as the name
6. Click "Generate"
7. Copy the 16-character password and use it in .env file

OpenAI API Key Setup (for AI Chatbot):
1. Go to https://platform.openai.com/api-keys
2. Sign up or log in to your OpenAI account
3. Click "Create new secret key"
4. Copy the API key and add it to your .env file as OPENAI_API_KEY
5. (Optional) Set OPENAI_MODEL to "gpt-4" for better responses, or "gpt-3.5-turbo" for faster/cheaper responses

Note: If OPENAI_API_KEY is not set, the chatbot will use a fallback rule-based system.
